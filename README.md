## 1. Меню

Меню утилиты открывается нажатием клавиши **INS** или **DEL**. Поддержка двух клавиш реализована для совместимости с нестандартными и компактными клавиатурами.

### 1.1. Требования к отображению

Для корректного отображения интерфейса установите режим окна **«Оконный без рамки»** в настройках лаунчера игры.

### 1.2. Точная настройка ползунков

Для пошаговой регулировки значения используйте **колёсико мыши**: наведите курсор на ползунок и прокручивайте вверх или вниз.

### 1.3. Защита интерфейса от захвата экрана

Интерфейс утилиты поддерживает скрытие от OBS, трансляций в Discord и инструментов создания скриншотов.

**Активация:**
1. Перейдите в раздел **Общее**.
2. Включите переключатель **«Защита от захвата интерфейса»**.

## 2. Конфигурации

Система управления конфигурациями позволяет сохранять, копировать и применять наборы настроек.

| Кнопка | Описание |
|--------|----------|
| **Save** | Сохраняет текущие настройки в активный профиль |
| **Copy** | Копирует конфигурацию в буфер обмена для передачи другому пользователю |
| **Paste** | Вставляет и применяет конфигурацию из буфера обмена |

> **Примечание:** продажа конфигураций **разрешена**.

## 3. Позиции кнопок и прогресс-баров

Все элементы интерфейса поддерживают ручную привязку к координатам экрана — совместимо с любым разрешением и соотношением сторон монитора.

### 3.1. Настройка позиции кнопки

> ⚠️ **Важно:** наводиться необходимо **исключительно на активную кнопку**. Если кнопка неактивна в момент сохранения — функция работать не будет.

1. Откройте панель выбора позиций.
2. Нажмите на нужную кнопку в панели (она должна быть активна).
3. Нажмите иконку **пипетки** в правом верхнем углу панели.
4. Наведите курсор на нужное место экрана.
5. Нажмите **Enter** для сохранения позиции.

### 3.2. Настройка позиции прогресс-бара

> ⛔ **Критически важно:** сохранять позицию необходимо **строго на цветной полоске** прогресс-бара. Позиция на фоне или пустой части шкалы — автоматическое использование предметов работать не будет.

1. Выберите нужный прогресс-бар (здоровье или голод) в панели позиций.
2. Определите пороговое значение шкалы, при котором бот должен использовать предмет.
3. Наведите курсор точно на **цветную полоску** в этой точке.
4. Нажмите **Enter** для сохранения позиции.

После настройки всех элементов панель выбора позиций можно закрыть.

## 4. Фитнес

> ⚠️ **Важно:** сначала настройте все параметры, и только после этого активируйте **главный контроллер**. Активация до завершения настройки приведёт к некорректной работе функций.

### 4.1. Переключатели

| Переключатель | Описание |
|---------------|----------|
| **Включённый** | Главный контроллер модуля. Включает или отключает работу алгоритма |
| **Автоматическая еда** | Автоматически использует еду из указанного слота при снижении голода. Требует настройки позиции прогресс-бара и номера слота |
| **Автоматическое лечение** | Автоматически использует аптечку из указанного слота при снижении здоровья. Требует настройки позиции прогресс-бара и номера слота |
| **Активный режим** | Предотвращает отключение от сервера по AFK-таймауту. Автоматически закрывает диалог проверки на бота |

### 4.2. Параметры скорости

| Параметр | Описание |
|----------|----------|
| **Скорость пробела** | Интервал между нажатиями клавиши Пробел (мс). Рекомендуется: пинг + 5 мс. Пример: пинг 30 мс → значение 35 мс |
| **Скорость кнопок** | Интервал между нажатиями кнопок тренажёра (мс). Рекомендуется: пинг + 5 мс |

### 4.3. Прочие параметры

| Параметр | Описание |
|----------|----------|
| **Категория** | Тип тренажёра для работы бота. Должна соответствовать активному тренажёру в игре |
| **Слот аптечек** | Номер слота быстрых действий (1–6), в котором находится аптечка |
| **Слот еды** | Номер слота быстрых действий (1–6), в котором находится еда |

## 6. Автоматический захват

### 6.1. Подготовка

1. В поле **«Задержка интернета»** укажите актуальный пинг до сервера.
2. Установите желаемую скорость потока (значение произвольно).
3. При необходимости включите **«Звук клавиатуры»** (необязательно — откаты с микрофоном в этом не нуждаются).

### 6.2. Отправка команды захвата

Заблаговременно введите в чат игры команду `/capture`. При наступлении тайминга программа самостоятельно нажмёт **Enter** и отправит команду.

### 6.3. Синхронизация системного времени

Для точного соблюдения тайминга синхронизируйте системные часы с эталонным сервером:

1. Нажмите **Win + R**, введите `timedate.cpl` и нажмите Enter.
2. Перейдите на вкладку **«Время по Интернету»**.
3. Нажмите **«Изменить параметры»**.
4. Убедитесь, что установлена галочка **«Синхронизировать с сервером времени в Интернете»**.
5. В поле сервера укажите `pool.ntp.org`.
6. Нажмите **«Обновить сейчас»** и подтвердите.

### 7.2. Telegram-бот

Встроенная функция удалённого управления через Telegram. После настройки управлять программой можно из любой точки мира без доступа к компьютеру.

**Порядок настройки:**

1. Откройте Telegram, перейдите к боту **@BotFather** и отправьте `/newbot`.
2. Введите произвольное отображаемое имя, например: `Malinovka Tools`.
3. Введите username в формате `название_bot`, например: `mtools_bot`.
4. Скопируйте выданный **токен** (строка вида `1234567890:AAAA...`).
5. Вставьте токен в соответствующее поле в разделе «Общее» и активируйте контроллер.
6. Откройте созданного бота в Telegram и отправьте `/start` — появится панель управления.

> ⚠️ **Важно:** храните токен в тайне. Любой, кто получит токен, сможет управлять вашим ботом и программой.
