# Malinovka Tools | FAQ

Настоящая документация содержит исчерпывающие инструкции по работе с Malinovka Tools. Перед обращением в службу технической поддержки убедительно просим вас внимательно ознакомиться с материалом. Если после повторного прочтения ваши вопросы останутся без ответа, свяжитесь с нашими специалистами.

# Глава первая "Меню"

Меню утилиты вызывается как нажатием клавиши INS, так и клавиши DEL. Это сделано для удобства пользователей с нестандартными или неполноценными клавиатурами. Для корректного отображения интерфейса в игре установите режим окна «Оконный без рамки». Данная настройка доступна в параметрах лаунчера игры.

Меню и интерфейс утилиты имеют возможность скрываться от скриншотов, демонстраций экрана в Discord и записывающих программ (например, OBS Studio).

Для активации данной функции перейдите в раздел «Общее» → «Защита от захвата интерфейса».

Для более точной настройки значения через ползунок вы можете воспользоваться колёсиком мыши. Просто наведите курсор на любую часть ползунка и прокручивайте колёсико вверх или вниз – значение будет изменяться с шагом, позволяющим задать нужный параметр максимально точно.

# Глава вторая "Конфигурации"

В Malinovka Tools реализована гибкая система управления конфигурациями. Для работы с настройками предусмотрены три основные кнопки:

- Save — сохранение текущих настроек;
- Copy — копирование конфигурации для передачи другому пользователю;
- Paste — вставка скопированной конфигурации в программу.

# Глава третья "Позиция кнопок и прогресс баров"

Для того чтобы каждый пользователь с любым монитором мог комфортно использовать Malinovka Tools, в программу добавлена панель выбора позиций кнопок. Вы можете настроить расположение элементов интерфейса под себя — независимо от разрешения экрана, диагонали или пропорций монитора.

![Позиция интерфейса](https://raw.githubusercontent.com/graffgriffin/malinovka-tools-faq/main/assets/images/buttons.png)

Чтобы задать позицию кнопки или прогресс-бара, выполните следующие шаги:

1. Нажмите на нужную кнопку на панели выбора позиций.
2. Нажмите на пипетку, которая находится в правом верхнем углу.
3. Наведите курсор на нужное место на экране.
4. Нажмите **Enter**, чтобы сохранить позицию.

> **⚠️ Важное предупреждение:**  
> Наводиться необходимо **исключительно на АКТИВНУЮ кнопку**.  
> Если вы сохраните позицию, когда кнопка неактивна, работать ничего **НЕ БУДЕТ**!

Чтобы задать позицию для прогресс-бара, выполните следующие шаги:

1. Выберите нужный прогресс-бар в панели выбора позиций.
2. Наведите курсор на позицию прогресс-бара.
3. Наводитесь на ту часть шкалы, до которой должна дойти полоска, чтобы бот автоматически использовал еду или аптечку.

> **⚠️ Критически важно:**  
> Сохранять позицию необходимо **строго на ЦВЕТНОЙ ПОЛОСКЕ** прогресс-бара!  
> Если сохранить позицию на фоне или пустой части шкалы — автоматическое использование предметов **НЕ БУДЕТ РАБОТАТЬ**!

Ниже показан пример, где лучше всего сохранять позицию:

![Пример](https://raw.githubusercontent.com/graffgriffin/malinovka-tools-faq/main/assets/images/hud.png)

После того как вы задали позиции для всех необходимых кнопок и прогресс-баров, можно закрыть панель выбора позиций.

# Глава четвёртая "Фитнес"

Для корректной работы утилиты соблюдайте следующий порядок действий:

> **⚠️ Важное правило:**  
> Сначала задайте все необходимые настройки, и только после этого включайте **главный контроллер**!  
> Если включить контроллер до настройки параметров, работа функций может быть некорректной.

Функции и их назначение:

- **Включённый** – главный тумблер для включения или отключения работы алгоритма;
- **Автоматическая еда** – автоматически использует еду из инвентаря при снижении голода. Требуется предварительно задать слот в настройках;
- **Автоматическое лечение** – автоматически использует аптечку из инвентаря при низком уровне здоровья. Требуется предварительно задать слот в настройках;
- **Активный режим** – предотвращает отключение с сервера из-за бездействия (AFK). Автоматически закрывает диалог с проверкой на бота при его появлении;


- **Скорость пробела** – задержка между нажатиями клавиши пробел в миллисекундах. Настраивается в зависимости от задержки интернета (например, при пинге 30 мс рекомендуется ставить 35 мс);
- **Скорость кнопок** – задержка между нажатиями основных кнопок в миллисекундах. Настраивается в зависимости от задержки интернета (например, при пинге 30 мс рекомендуется ставить 35 мс);


- В **«Категории»** выбирается нужный тренажёр для работы бота;


- **Слот аптечек** – выбор слота быстрых действий, в котором находится аптечка;
- **Слот еды** – выбор слота быстрых действий, в котором находится еда;

# Глава пятая "Переподключение"

- Во вкладке присутствует один контроллер для включения и выключения функции блокировки. При активации программа блокирует сетевые пакеты только для процесса `malinovka_game.exe`. Это позволяет сохранять стабильное интернет-соединение для Discord, браузеров и других программ.

# Глава шестая "Автоматический захват"

Перед запуском выполните следующие настройки:

1. Укажите свой пинг в "Задержка интернета";
2. Установите желаемую скорость потока (можно любую);
3. При желании включите звук клавиатуры (необязательно, так как откаты с микрофоном не требуют;

Для отправки команды захвата приготовьте в чате `/capture`. Программа только нажмёт клавишу Enter за вас.

Для отправки команды в идеальный тайминг необходимо синхронизировать время на компьютере:

1. Нажмите сочетание клавиш `Win + R`;
2. Введите `timedate.cpl` и нажмите `Enter`;
3. Перейдите на вкладку **Время по Интернету**;
4. Нажмите **Изменить параметры**;
5. Убедитесь, что стоит галочка **Синхронизировать с сервером времени в Интернете**;
6. Смените сервер на `pool.ntp.org`;
7. Нажмите **Обновить сейчас**;

# Глава седьмая "Общее"

Функции и их назначение:

**- **Автоматическая авторизация** – включает автоматический вход в приложение, чтобы каждый раз не вводить ключ доступа вручную;**  
**- **Защита от захвата интерфейса** – предотвращает перехват окна приложения. Интерфейс не будет отображаться в OBS, демонстрациях экрана в Discord и на скриншотах;**  
**- **Звуки интерфейса** – управляет звуковым сопровождением элементов UI (клики, уведомления и системные сигналы).**

**Телеграм-бот: что это, как настроить и почему это удобно?**

В приложение встроена функция удалённого управления через Telegram-бота.  
Это позволяет контролировать программу из любой точки мира, где есть интернет, без необходимости открывать само приложение.

**Как настроить бота**

Чтобы начать использовать эту возможность, вам нужно перейти к официальному боту Telegram – `@BotFather` и отправить команду `/newbot`.

После этого вы получите сообщение:

> `Alright, a new bot. How are we going to call it? Please choose a name for your bot.`

Вам нужно придумать имя для своего бота, например `Malinovka Tools` (можно любое).

Затем бот напишет:

> `Good. Now let's choose a username for your bot. It must end in 'bot'. Like this, for example: TetrisBot or tetris_bot.`

Теперь нужно придумать **username** для бота в формате `{название}_bot`, например `mtools_bot`.

После успешного создания появится сообщение:

> `Done! Congratulations on your new bot. You will find it at t.me/mtools_bot. You can now add a description, about section and profile picture for your bot, see /help for a list of commands. By the way, when you've finished creating your cool bot, ping our Bot Support if you want a better username for it. Just make sure the bot is fully operational before you do this.`
>
> `Use this token to access the HTTP API:`
> `1222222222:AAAA1-aAa-aa3AaA-a5AAAaAA1Aa0aAaAA`
>
> `Keep your token secure and store it safely, it can be used by anyone to control your bot.`
>
> `For a description of the Bot API, see this page: https://core.telegram.org/bots/api`

---

**Завершение настройки**

1. Скопируйте токен (набор букв и цифр):  
   `1222222222:AAAA1-aAa-aa3AaA-a5AAAaAA1Aa0aAaAA`

2. Вставьте его в соответствующее поле в программе и включите контроллер.

3. Перейдите по ссылке `t.me/mtools_bot` (или введите имя бота в поиске Telegram).

4. Отправьте команду `/start` – перед вами появится панель управления.
